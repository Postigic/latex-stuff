\begin{claim}
    The robot can never reach $(1,1)$.
\end{claim}

\begin{proof}
    Define the invariant $f(x,y) = 4x + y$ and observe that $f(0,0) = 0 \equiv 0 \pmod{7}$.

    \medskip
    We show that every move preserves $f(x,y) \equiv 0 \pmod 7$. For each allowed displacement:

    \[
    \begin{aligned}
    (+2,-1):\quad & f(x+2,\, y-1) = f(x,y) + 7 \equiv 0 \pmod{7} \\
    (-2,+1):\quad & f(x-2,\, y+1) = f(x,y) - 7 \equiv 0 \pmod{7} \\
    (+1,+3):\quad & f(x+1,\, y+3) = f(x,y) + 7 \equiv 0 \pmod{7} \\
    (-1,-3):\quad & f(x-1,\, y-3) = f(x,y) - 7 \equiv 0 \pmod{7}
    \end{aligned}
    \]

    In all cases the invariant is preserved, so by induction every reachable
    position $(x,y)$ satisfies $f(x,y) \equiv 0 \pmod{7}$.

    \medskip
    However, $f(1,1) = 4 + 1 = 5 \not\equiv 0 \pmod{7}$,
    so $(1,1)$ is unreachable.
\end{proof}
